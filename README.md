# barista.build

Marketing site and documentation for [Barista](https://github.com/buildwithbarista/barista) —
a fast, Maven-compatible build tool for the JVM.

This repository powers:

- **barista.build** — project landing page, docs, install instructions
- **bench.barista.build** — published benchmark methodology and results
- **barista.build/schema/** — hosted JSON schemas (lockfile, output formats)
